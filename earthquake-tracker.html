Earthquake Tracker

About this Project

Tracking, reporting, and visualizing data is an integral part of using the internet. Websites, like this single page application, help make sharing information more accessible and can give the user the opportunity to interact with the data as well. This project incorporates the latest earthquakes as reported by the United States Geological Survey and Google Maps to show users where these instaces took place.

Javascript HTML CSS Bootstrap Google Maps USGS API
GitHub Repo
Earthquake Tracker App

Discovering the Latest

When the user loads the page, the most recent events are displayed on to the left of the map. Each event shows the location, date, time, magnitude, longitude, and latitude. Each time the page refreshes, it will load the three latest events as reported by USGS.

Interacting with the Data

Users can click on one of the three most recent events to load its location on Google Maps. Visualizing the data is vital to providing users with perspective on these seismic events.

What's Next

This app can be expanded on by adding additional user preferences and search capabilities. Some examples include saving a user's location and alert them when an earthwuake is within a specific distance.
